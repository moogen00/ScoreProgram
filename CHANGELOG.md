## [v1.1.0] - 2026-01-16 (Data Safety & Deployment)
### 추가됨 (Added)
- **데이터 삭제 안전장치**: 연도, 종목, 참가자, 채점 기준 삭제 시 점수 데이터가 존재하면 경고창 표시.
- **심사위원 익명화 (Anonymization)**: 점수 기록이 있는 심사위원 삭제 시, 데이터를 지우지 않고 이름을 '알수 없음'으로 변경하여 점수 보존.
- **DB 초기화 안전장치**: '데이터베이스 완전 초기화' 시 2단계 확인(Double Confirmation) 절차 도입.
- **협업 가이드**: README.md에 Windows-Mac 멀티 디바이스 개발 워크플로우 가이드 추가.

### 변경됨 (Changed)
- **점수 제출 방식 개선**: 자동 저장(Auto-save) 방식에서 '저장 및 제출(Save & Submit)' 버튼을 누르는 수동 제출 방식으로 변경. (심사위원의 실수 방지)
- **배포**: Firebase Hosting으로 실서버 배포 완료.

### 수정됨 (Fixed)
- **관리자 삭제 경고**: 최고 관리자(Root Admin)가 다른 관리자를 삭제할 때 경고창이 뜨지 않던 문제 해결.

## [v1.0.1] - 2026-01-15 (Admin & UI Improvements)
### 추가됨 (Added)
- **전체 참가자 현황 탭**: 관리자 패널에 연도별 전체 참가자를 모아보는 탭 추가.
- **순위 시각화**: 스코어보드에서 1, 2, 3위 등 상위권 참가자에게 색상(Rose/Gold) 하이라이트 적용.

### 수정됨 (Fixed)
- **로그인 오류**: 심사위원 구글 로그인 시 "Unregistered User" 오류가 발생하는 문제 해결.
- **파일 내보내기**: 데이터 백업(JSON) 및 엑셀 다운로드 기능이 작동하지 않던 문제 수정.
- **화면 깨짐**: 관리자 패널 진입 시 간헐적으로 검은 화면(Black Screen)이 뜨던 렌더링 오류 수정.

## [v1.0.0] - Initial Release
### 기능 (Exisiting Features)
- **실시간 채점**: 심사위원 점수 입력 시 실시간 서버 동기화.
- **관리자 패널**: 연도/종목/참가자/심사위원 관리 기능 (CRUD).
- **대쉬보드/리더보드**: 실시간 순위 및 점수 집계 화면.
- **관람객 모드**: QR 코드를 통한 일반 관람객 접속 지원.
